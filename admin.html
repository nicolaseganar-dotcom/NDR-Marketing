⚙️ Configurar Supabase
Sigue estos pasos una sola vez para activar el panel. Después todo funciona solo.
1
Crea una cuenta gratis en supabase.com

Ve a supabase.com → New project → dale un nombre (ej: "ndr-marketing") → elige región São Paulo (más cercana a Chile).

2
Crea la tabla site_config

En el dashboard de Supabase → SQL Editor → pega y ejecuta este SQL:

CREATE TABLE site_config ( key TEXT PRIMARY KEY, value TEXT ); -- Valores iniciales INSERT INTO site_config (key, value) VALUES ('pixel_id', ''), ('ga4_id', ''), ('whatsapp_number', '56926879731'), ('whatsapp_message', 'Hola Nicolás, vengo de tu web y me gustaría saber más.'), ('calendly_url', 'https://calendly.com/ndrmarketing'), ('stat_revenue', '$191M+'), ('stat_roas', '28x'), ('stat_years', '5+'), ('stat_clients', '20+'), ('profile_photo', ''), ('og_image_paid_media', ''), ('og_image_diseno_web', ''), ('og_image_quienes_somos', ''), ('og_image_portafolio', ''), ('og_image_precios_web', ''); -- Permitir lectura pública (para config-loader.js) ALTER TABLE site_config ENABLE ROW LEVEL SECURITY; CREATE POLICY "public read" ON site_config FOR SELECT USING (true); CREATE POLICY "authed write" ON site_config FOR ALL USING (auth.role() = 'authenticated');
3
Crea el Storage bucket "assets"

En Supabase → Storage → New bucket → nombre: assets → marca "Public bucket" → Create.

4
Copia tus credenciales

En Supabase → Settings → API → copia Project URL y anon public key.

5
Pega las credenciales en config-loader.js

Abre el archivo config-loader.js y reemplaza las dos constantes al inicio:

const SUPABASE_URL = 'https://xxxx.supabase.co'; const SUPABASE_ANON_KEY = 'eyJhbGci...';
6
También pégalas aquí para usar el panel
